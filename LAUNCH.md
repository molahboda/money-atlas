# 머니 아틀라스 — 수익화 런칭 체크리스트

> 사이트와 수익 장치는 전부 준비되어 있습니다. 아래에서 **계정이 필요한 것만 본인이** 만들고,
> 발급된 링크/ID를 `js/config.js` 에 붙여넣으면 사이트에 자동으로 켜집니다.

## STEP 1 — 오늘 (30분, 전부 무료)

| # | 할 일 | 소요 | 결과물 → 붙여넣을 곳 |
|---|---|---|---|
| 1 | [GitHub](https://github.com) 가입 → 터미널에서 `brew install gh` → `gh auth login` | 10분 | 로그인 후 Claude에게 "배포해줘" — repo 생성·push·Pages 설정을 대신 처리 |
| 2 | [Buy Me a Coffee](https://buymeacoffee.com) 또는 토스 후원 링크 생성 | 5분 | `config.js` → `supportLink` |
| 3 | [스티비](https://stibee.com) 무료 가입 → 구독 폼 생성 | 10분 | `config.js` → `premiumWaitlistUrl` (프리미엄 대기명단 = 첫 자산) |

## STEP 2 — 이번 주

- **쿠팡파트너스** (즉시 승인): 배너 HTML → `config.js` → `coupangPartnersHtml` — 애드센스 승인 전 임시 수익원
- **구글 애드센스** 신청 (심사 1~4주): 승인 후 `ca-pub-…` → `config.js` → `adsenseClient` — 승인율을 높이려면 아래 "SEO 페이지 분리"를 먼저 할 것
- (선택) 도메인 구매 ~2만원/년 — 애드센스 심사와 브랜딩에 유리

## STEP 3 — 트래픽 (수익의 전제)

수익 = 트래픽 × 전환. **첫 30일의 목표는 돈이 아니라 방문자입니다.**
- 역사 14국면·패턴 9개를 개별 URL 페이지로 분리 (검색 유입 + 애드센스 심사 통과의 핵심) — Claude에게 요청
- "오늘의 나침반" 공유 카드 → X·커뮤니티 배포
- 급락일 = 뉴스 브리핑 캡처가 가장 잘 퍼지는 날

## STEP 4 — 유료 멤버십 (4~8주 뒤, 대기명단이 200명쯤 모이면)

1. 사업자등록 (홈택스, 무료) → 통신판매업 신고 (구청, ~4만원)
2. 유사투자자문업 해당 여부 변호사 검토 1회 → 필요 시 금감원 신고
3. 결제: 스티비 유료 뉴스레터 or 토스페이먼츠 연동
4. 이용약관·개인정보처리방침·환불정책 게시

## ⚠️ 법적 최소선 (공개 전 확인)

- 무료 공개 단계: 현행 구성은 회색지대의 안전한 쪽이나, **트래픽이 커지기 전에 데이터 소스를
  공식 API(ECOS·FRED)/TradingView 위젯으로 교체**할 것 (야후 비공식 API·뉴스 제목 리스크)
- 상시 금지: 특정 종목 리딩 ✕ · 1:1 상담 ✕ · 유료 양방향 채팅방 ✕ · 수익 보장 문구 ✕

## 💰 현실적인 수익 타임라인

| 시점 | 수익원 | 현실적 기대치 |
|---|---|---|
| 1주차 | 후원 + 쿠팡 | 0~몇만 원 (트래픽 없으면 0) |
| 1~2개월 | 애드센스 (금융 니치 RPM 높음) | 월 1만 PV당 3~10만 원 |
| 3개월+ | 프리미엄 멤버십 (월 7,900원안) | 대기명단 전환 10~30% → 유료 100명 = 월 ~76만 원 |
